# Selenium TestNG Data-Driven Framework

## Overview

This project is a scalable UI automation framework built using Selenium WebDriver, TestNG, and Java. It follows the Page Object Model (POM) design pattern and supports data-driven testing using TestNG DataProvider.

## Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven

## Features

* Page Object Model (POM)
* Data-driven testing using @DataProvider
* Reusable and maintainable code structure
* Maven-based project
* Cross-browser support (can be extended)

## Project Structure

```
src
 ├── pages
 ├── tests
 ├── base
 └── utils
```

## How to Run

1. Clone the repository
2. Navigate to project folder
3. Run:

```
mvn clean test
```
